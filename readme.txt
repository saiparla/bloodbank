Blood Bank Project
This project is a web application designed to streamline interactions between blood donors and recipients. It provides an efficient platform for managing and accessing donor information.
________________________________________
Features
•	User Registration and Authentication:
Users can register and log in to access application features securely.
•	Donor Information Management:
Users can add, update, and view donor details.
•	Search Functionality:
Easily search for donors based on blood group, location, and availability.
•	Responsive UI:
A user-friendly and adaptive interface built using MUI (Material-UI).
________________________________________
Technologies Used
Frontend
•	Framework: MUI (Material-UI)
•	Languages: React.js, JavaScript, HTML, CSS
Backend
•	Framework: Node.js with Express.js
Database
•	Database: MySQL
________________________________________
Installation and Setup
1.	Clone the repository: git clone https://github.com/saiparla/bloodbank.git
2.  Navigate to the project directory: cd bloodbank
3.	Install dependencies: npm install
4.	Set up MySQL database:
    o	Create a new database in MySQL.
    o	Import the provided SQL file (/database_schema.sql) into the database.
    o	Update the .env file with the following: 
    o	DB_HOST=localhost
    o	DB_USER=your_mysql_user
    o	DB_PASSWORD=your_mysql_password
    o	DB_NAME=your_database_name
    o	PORT=3000
    o	SESSION_SECRET=your_session_secret
5.	Run the application: npm start
The application will be available at http://localhost:3000.
________________________________________
Usage
1.	Register:
  Create an account to access donor and recipient features.
2.	Log In:
  Log in using your credentials to access the dashboard.
3.	Manage Donors:
  Add and  view blood donor information.
4.	Search for Donors:
  Use the search functionality to filter results by blood group, location, or other criteria.
